# Loyalty Rewards Portal 🎁

A sleek and beginner-friendly web portal for managing customer loyalty rewards, built using HTML, CSS, and JavaScript.

## 🌟 Features

- 🎉 Elegant homepage with carousel banner
- 💬 Chatbot assistant for instant support
- 📋 Simple reward tracking interface
- 🔐 Login & Signup form UI (frontend only)
- 📱 Responsive layout for all screen sizes
- ❓ FAQ Section with Q&A dropdown

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Design**: CSS Flexbox, Transitions, Carousel, and Animations
- **Chatbot**: Simulated using JavaScript prompts and responses
